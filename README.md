# Node.js Starter Project

A minimal Node.js starter project with Express.

## Features

- ES Modules support
- Express server
- Environment variables with dotenv
- Watch mode for development
- Basic project structure

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. For production:
   ```bash
   npm start
   ```
